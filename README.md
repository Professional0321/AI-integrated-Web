## AI-integrated Web App for SERVIMATT


A lightweight AI chat-style prompt app with clean UI, resilient fetch logic, local history, and a provider switch for OpenAI or Hugging Face.


## Features
- Prompt input + submit button
- Fetching from OpenAI
- Dynamic result display
- Loading + error states
- Local chat history + Clear button


## Stack
  React 19, TypeScript, Vite


## Setup
**Install**
```bash
npm i
```
**Run**
```bash
npm start
```
